# ElektronReact

Bu proje basit bir Electron ve React entegrasyonunu gösterir. Uygulama içerisinde
hamburger menüsü ile "Anasayfa", "Hakkımızda" ve "Bilgiler" sayfalarına
erişebileceğiniz örnek bir yapı bulunmaktadır.

Uygulamayı çalıştırmak için:

```bash
npm install
npm start
```

Geliştirme sürecinde `npm run start:dev` komutu kullanılabilir.
