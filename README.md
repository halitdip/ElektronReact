# ElektronReact

Electron ve React entegrasyonunu gosterir. Arayuz
elemanlari icin guncel ve yaygin bir kutuphane olan **MUI** (`@mui/material`)
kullanilmaktadir. Hamburger menusu ve sayfa gecisleri bu kutuphane yardimiyla
saglanmistir.
`react-router-dom` ile coklu sayfa yapisi olusturulmustur.
 
Uygulamayi calistirmak icin:

```bash
npm install
npm start
```

Geliştirme sürecinde ilk çalıştırmada `npx electron-rebuild` sonra`npm run start:dev` komutu kullanılabilir.
Console mesajlarini azaltmak icin `DEBUG=1 npm run start:dev` komutu ile
gelistirme araclari acilabilir.
